# 용어집

> **TL;DR** — 이 저장소에서 쓰는 도메인·기술 용어의 뜻을 정리합니다.

## 도메인

- **진료과 (department)** — 안과·내과·소아과 같은 병원의 과. `data.json`의 `departments` 배열 한 항목에 대응합니다.
- **대기 시간 (waitMinutes)** — 해당 진료과의 예상 대기 시간(분). 화면에는 "대기 시간 약 N분"으로 표시.
- **대기 인원 (waitingCount)** — 해당 진료과에서 대기 중인 환자 수(명).
- **마지막 갱신 시각 (updatedAt)** — 데이터가 만들어진 시각. ISO 8601 문자열이며 헤더에 한국어 형식으로 표시됩니다.

## 기술

- **데이터 소스 (DATA_SOURCE)** — `app.js` 최상단 상수. 데이터를 가져올 위치를 가리킵니다. 현재는 `./data.json`이고, 실제 백엔드 연동 시 이 한 줄을 엔드포인트 URL로 교체합니다.
- **빌드리스 정적 사이트 (build-less)** — 번들러·패키지 매니저·서버 코드 없이 정적 파일만으로 동작하는 구성. 브라우저로 바로 열거나 단순 정적 서버로 띄웁니다.
- **카드 (card)** — 한 진료과의 대기 정보를 담은 화면 단위(`.department-card`). `app.js`가 진료과마다 하나씩 생성합니다.
